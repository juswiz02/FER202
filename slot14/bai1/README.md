# Slot 14 - Bài 1: Routing Cơ Bản

## Yêu cầu
- Tạo 3 component: Home, Products, Contact
- Cấu hình route:
  - `/` → Home
  - `/san-pham` → Products
  - `/lien-he` → Contact
- Tạo Navbar sử dụng NavLink, highlight trang hiện tại

## Cài đặt
```bash
npm install
npm install react-router-dom@6
npm start
```
